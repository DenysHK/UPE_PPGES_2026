"DHK_C3S MD ANUAIS ACIMA DE 1.5C.py" é o gerador do gráfico que mostra a elevação da temperatura média global em relação ao período pré-industrial (1850-1900) utilizando a base de dados harmonizada do C3S.

 "SFT_Pandey2021_DWSIM.py" é uma versão com bastante comentários do Script Python que usei no "SFT  Pandey2021.dwxmz" para implementação da cinética LHHW e balanço de massa "externamente" ao DWSIM. 

"ALPHA.py" é o gerador do gráfico da distribuição ASF (Probabilidade de crescimento da cadeia ALPHA x Fração mássica (wt%) para C1, C2-C4, C5-C11, C12-C20, C20-C30  e C31+.
